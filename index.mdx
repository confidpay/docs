# ConfidPay

Privacy-by-design confidential payroll and treasury rail for DAOs and Web3 teams.

Built on **Fhenix CoFHE** + **Privara SDK**.

## Overview

ConfidPay enables encrypted payroll processing where salaries, recipients, vesting schedules, and milestone conditions remain encrypted during computation — no plaintext values ever touch the chain.

**Key properties:**
- Amounts, recipients, and conditions never visible on-chain
- Programmable release rules (time-based, milestone-gated, vesting)
- Selective disclosure to intended recipients only
- Full EVM composability

## Problem

Public blockchains expose everything. DAOs and Web3 teams cannot run transparent rails due to:

- Social attacks and doxxing risks
- MEV front-running on recurring payments
- Compliance barriers for institutions

**The challenge:** Maintain programmability while keeping data encrypted end-to-end.

## Solution

FHE enables computation on encrypted data without decryption:

```solidity
// Salary stored as encrypted integer — never readable on-chain
euint128 encryptedAmount;

// Contract can still compute:
ebool canRelease = FHE.lte(encryptedNextRelease, currentTime);
euint128 payment = FHE.ifThenElse(canRelease, encryptedAmount, 0);
```
