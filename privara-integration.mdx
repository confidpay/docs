# ReineiraOS Integration

ConfidPay uses ReineiraOS for confidential escrow infrastructure and insurance coverage on payment flows.

## Overview

```
┌─────────────────────────────────────────────────────────────┐
│                      ConfidPay.sol                          │
│  • Encrypted payroll logic                                  │
│  • ACL management                                           │
│  • Release conditions                                       │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│                    ReineiraOS Layer                        │
│  • IConfidentialEscrow - Hold funds until conditions met   │
│  • IInsuranceManager - Coverage on escrow                  │
│  • IFHERC20 - Confidential USDC transfers                  │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│                   USDC (on Arbitrum Sepolia)               │
└─────────────────────────────────────────────────────────────┘
```

## Payment Flow

### Before (Direct Transfer)

```
Admin → createPayroll() → salary stored encrypted
Employee → claimPayment() → confidentialToken.transfer() → Employee gets tokens
```

### After (Escrow Flow)

```
Admin → createPayroll() → salary stored encrypted
Admin → createPaymentEscrow() → Escrow created (off-chain)
Admin → fundEscrow() → Funds escrow + insurance auto-attached
Employee → getMyPendingEscrows() → View pending escrows
Employee → redeem() via SDK → Employee gets tokens
```

## Contract Interfaces

### IConfidentialEscrow

```solidity
interface IConfidentialEscrow {
    function create(
        address owner,
        uint256 amount,
        address resolver,
        bytes calldata resolverData
    ) external returns (uint256);
    
    function fund(uint256 escrowId, uint256 amount) external;
    function redeem(uint256 escrowId) external;
    function exists(uint256 escrowId) external view returns (bool);
    function isFunded(uint256 escrowId) external view returns (bool);
    function isRedeemable(uint256 escrowId) external view returns (bool);
    function paymentToken() external view returns (address);
}
```

### IInsuranceManager

```solidity
interface IInsuranceManager {
    function purchaseCoverage(
        address pool,

        uint256 escrowId,

    function getCoverageStatus(uint256 coverageId) external view returns (uint8);

```

## ConfidPay Integration

### New State Variables

```solidity
IFHERC20 public confidentialToken;
IConfidentialEscrow public escrowContract;
IInsuranceManager public insuranceManager;
address public insurancePoolId;      // Changed from uint256 → address
address public insurancePolicy;
uint256 public coverageExpirySeconds;
```

### New Functions

| Function | Purpose |
|----------|---------|
| `configureInsurance(address pool, address policy, uint256 expirySeconds)` | Admin sets insurance pool/policy |
| `createPaymentEscrow(address employee, uint256 amount)` | Creates escrow for employee |
| `fundEscrow(address employee, uint256 escrowId, uint256 amount)` | Funds escrow + auto-purchases insurance |
| `getMyPendingEscrows()` | Employee views pending escrows |
| `markEscrowClaimed(uint256 escrowId)` | Marks escrow as claimed |
| `getEscrowContract()` | Returns escrow contract address |
| `getInsuranceConfig()` | Returns insurance settings |

### Insurance Auto-Attachment

```solidity
function fundEscrow(
    address employee,
    uint256 escrowId,
    uint256 amount
) external onlyAdmin {
    escrowContract.fund(escrowId, amount);
    
    if (
        address(insuranceManager) != address(0) &&
        insurancePolicy != address(0)
    ) {
        uint256 expiry = block.timestamp + coverageExpirySeconds;
        insuranceManager.purchaseCoverage(
            insurancePoolId,
            insurancePolicy,
            escrowId,
            amount,
            expiry
        );
    }
}
```
