# Privara SDK Integration

Privara SDK provides the payment rail layer for confidential USDC transfers in ConfidPay.

## Overview

```
┌─────────────────────────────────────────────────────────────┐
│                      ConfidPay.sol                          │
│  • Encrypted payroll logic                                  │
│  • ACL management                                           │
│  • Release conditions                                       │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Privara SDK Layer                         │
│  • ConfidentialTransfer                                     │
│  • Escrow                                                   │
│  • Programmable release                                     │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                     USDC (on Base/Arbitrum)                  │
└─────────────────────────────────────────────────────────────┘
```

## Core Patterns

### Confidential Transfer

```typescript
import { ConfidentialTransfer, TransferType } from '@reineiralabs/sdk';

const transfer = new ConfidentialTransfer({
  token: 'USDC',
  amount: encryptedAmount,
  recipient: encryptedRecipient,
  type: TransferType.PAYROLL,
  conditions: encryptedConditions,
});

const tx = await transfer.build();
await tx.wait();
```

### Escrow Pattern

```typescript
import { Escrow } from '@reineiralabs/sdk';

const escrow = new Escrow({
  token: 'USDC',
  depositedAmount: encryptedAmount,
  beneficiary: encryptedRecipient,
  releaseCondition: encryptedMilestoneHash,
  timeout: encryptedTimestamp,
});

// Deposit funds
await escrow.deposit({ value: depositAmount });

// Release when conditions met
await escrow.release();
```

### Programmable Release

```typescript
import { ProgrammablePayment } from '@reineiralabs/sdk';

const payment = new ProgrammablePayment({
  token: 'USDC',
  amount: encryptedSalary,
  recipient: encryptedRecipient,
  schedule: [
    {
      unlockTime: encryptedNextMonth,
      percentage: encryptedVestingRate,
    },
  ],
  milestoneConditions: encryptedMilestones,
});

await payment.create();
await payment.executeReleases();
```

## React Integration

### usePrivaraTransfer Hook

```typescript
import { usePrivaraTransfer } from '@reineiralabs/sdk/react';

function PayrollDashboard() {
  const { 
    createTransfer, 
    executeRelease, 
    getEncryptedBalance,
    isLoading, 
    error 
  } = usePrivaraTransfer({
    token: 'USDC',
    chainId: 84532, // Base Sepolia
  });

  const handleRelease = async (recipient: string) => {
    const encryptedRecipient = await encrypt(recipient);
    await executeRelease({
      recipient: encryptedRecipient,
      conditions: encryptedMilestoneHash,
    });
  };

  return (
    <button onClick={() => handleRelease(recipient)}>
      Release Payment
    </button>
  );
}
```

import { useEncryptedBalance } from '@reineiralabs/sdk/react';

## ConfidPay + Privara Integration

### Complete Flow

```typescript
import { useEncrypt, useWrite, useDecrypt } from '@cofhe/react';
import { ConfidentialTransfer } from '@reineiralabs/sdk';

async function createConfidentialPayroll(
  recipient: string,
  amount: bigint,
  vestingRate: number
) {
  const { encrypt } = useEncrypt();
  const { write } = useWrite();

  const encryptedAmount = await encrypt(amount);
  const encryptedRecipient = await encrypt(recipient);
  const encryptedVestingRate = await encrypt(vestingRate);
  const encryptedNextRelease = await encrypt(
    Math.floor(Date.now() / 1000) + 30 * 24 * 60 * 60
  );

  // 1. Create payroll in ConfidPay
  await write('createPayroll', [
    recipient,
    encryptedAmount,
    encryptedNextRelease,
    encryptedVestingRate,
    0,
  ]);

  // 2. Create confidential transfer via Privara
  const transfer = new ConfidentialTransfer({
    token: 'USDC',
    amount: encryptedAmount,
    recipient: encryptedRecipient,
    type: TransferType.PAYROLL,
  });

  await transfer.build().then(tx => tx.wait());
}

async function releasePayment(recipient: string) {
  const { write, wait } = useWrite();
  
  // 1. Trigger release in ConfidPay
  const tx = await write('releasePayment', [recipient]);
  await wait(tx);
  
  // 2. Execute confidential transfer via Privara
  const { decrypt } = useDecrypt();
  const encryptedBalance = await decrypt(/* encrypted balance */);
  
  const transfer = new ConfidentialTransfer({
    token: 'USDC',
    amount: encryptedBalance,
    recipient: await encrypt(recipient),
    type: TransferType.RELEASE,
  });

  await transfer.execute();
}
```

## Error Handling

```typescript
import { PrivaraError, ErrorCode } from '@reineiralabs/sdk';

try {
  await transfer.execute();
} catch (error) {
  if (error instanceof PrivaraError) {
    switch (error.code) {
      case ErrorCode.INSUFFICIENT_BALANCE:
        console.log('Insufficient encrypted balance');
        break;
      case ErrorCode.CONDITIONS_NOT_MET:
        console.log('Release conditions not satisfied');
        break;
      case ErrorCode.DECRYPTION_FAILED:
        console.log('Unable to decrypt transfer data');
        break;
    }
  }
}
```

## Best Practices

1. **Always encrypt client-side** before passing to SDK
2. **Set appropriate ACL** for all encrypted fields
3. **Handle errors gracefully** with proper error codes
4. **Use Testnet first** to verify integration before Mainnet
5. **Monitor gas costs** — encrypted operations cost more than plaintext
